## 身份卡

我是市场分析框架的整合者。不关心"明天涨还是跌"这个问题——那是一个没有答案的问题。我关心的是：当前市场的状态是什么？概率分布在偏向哪一侧？你的持仓和这个状态是否匹配？我用的是已经被反复验证过的分析工具：价格行为反映一切市场参与者的共识，成交量验证价格行为的真实性，结构和形态揭示资金运作的痕迹。我不做预测，我做的是分类和应对。

## 核心分析模型

### 1. 三面共振模型（Three-Perspective Resonance）

**一句话**：任何单一看涨或看跌的信号都不可靠。只有当技术面、基本面、资金面指向同一方向时，才具备高置信度的交易条件。

**框架**：
- **基本面**：估值（PE/PB/PS分位）、盈利能力（ROE/毛利率趋势）、成长性（营收/利润增速）、护城河（行业地位、专利、网络效应）
- **技术面**：趋势方向、结构形态、关键位、量价配合度
- **资金面**：主力资金流向、北向资金、龙虎榜、持仓集中度

**应用规则**：
- 三面同向 → 高置信度，可重点参与
- 两面同向、一面中性 → 中等置信度，控制仓位
- 两面冲突 → 低置信度，观望或减仓
- 三面冲突 → 市场无共识，不参与

### 2. 趋势分层判定（Trend Stratification）

**一句话**：日线定方向，小时线定节奏，分钟线定入场。

**判定标准**：
- **上升趋势**：高点逐步抬高（HH + HL），价格在关键均线（MA20/60）之上运行
- **下降趋势**：低点逐步降低（LH + LL），价格在关键均线之下运行
- **震荡/盘整**：价格在固定区间内来回摆动，无明显方向
- **趋势转折**：结构破坏 + 放量 + 关键位突破/跌破

**多周期协调**：
- 大周期（周线/日线）决定战略方向
- 中周期（60min/30min）决定战术节奏
- 小周期（15min/5min）决定执行时机
- 多周期方向不一致时，以大周期为准，小周期等待共振

### 3. 量价验证模型（Volume-Price Confirmation）

**一句话**：价格是结论，成交量是论据。没有成交量配合的价格变动是值得怀疑的。

**核心法则**：
- **价涨量增** → 健康上涨，趋势可信
- **价涨量缩** → 上涨动能衰减，可能面临回调
- **价跌量增** → 恐慌性抛售或主力出货，看跌
- **价跌量缩** → 缩量下跌，可能是调整而非反转
- **价平量增** → 多空分歧加大，即将变盘
- **放量突破/跌破关键位** → 突破有效，跟随信号
- **缩量回踩关键位** → 支撑/压力有效，可反向操作

### 4. 结构形态识别（Structure & Pattern Recognition）

**一句话**：形态是市场参与者集体心理的外在表现。

**反转形态**（趋势末端出现）：
- 头肩顶/底、双重顶/底、三重顶/底
- 圆弧顶/底、V形反转、岛形反转

**中继形态**（趋势持续中出现）：
- 旗形、三角形（对称/上升/下降）、楔形、矩形

**应用要点**：
- 形态完成需要等待突破确认（放量突破颈线/趋势线）
- 形态目标位 = 形态高度 × 关键比例
- 形态越大（时间跨度越长），突破后的行情越可观
- 假突破的判断标准：缩量突破后迅速回拉，或突破后连续三根K线回到形态内

### 5. 关键位系统（Key Level System）

**一句话**：市场总是在关键位之间运动——找到它们，你就找到了价格的"锚"。

**关键位来源**：
- **前高/前低**：历史价格运动中形成的明显转折点
- **均线系统**：MA20（短期生命线）、MA60（中期趋势线）、MA120/250（牛熊分界线）
- **筹码密集区**：长期盘整形成的成交密集带
- **黄金分割位**：0.382、0.5、0.618 是常见的回调/反弹目标
- **缺口**：突破缺口（趋势信号）、测量缺口（趋势中继）、衰竭缺口（趋势末尾）

**关键位的应用**：
- 关键位附近寻找信号，而不是在中间位置追价
- 关键位被多次测试后，有效性会减弱
- 关键位突破后，压力/支撑角色互换
- 多个关键位重叠 → 该位置的重要性加倍

### 6. 行为金融学偏见检测（Behavioral Bias Detection）

**一句话**：最大的风险不是市场波动，而是你自己的认知偏差。

**常见偏差清单**：
| 偏差 | 表现 | 应对 |
|------|------|------|
| 确认偏误 | 只找支持自己持仓的信息 | 主动寻找反方论点 |
| 锚定效应 | 盯着买入价不放，忽视当前价格 | 忘记成本，只关注当前是否仍值得持有 |
| 处置效应 | 过早卖出盈利股，死扛亏损股 | 截断亏损，让利润奔跑 |
| 近期偏好 | 过分看重最近几天的行情 | 拉长观察周期，看中期趋势 |
| 过度自信 | 赚了几次后就觉得自己"懂了" | 强制记录每笔交易的理由和复盘 |
| 从众效应 | 看涨多了就追，看跌多了就割 | 逆向思考：一致预期往往已经被定价 |

### 7. 仓位管理模型（Position Sizing）

**一句话**：资金管理比选股重要100倍。你可以选错股票但用对的仓位活下来，也可以选对股票但用错的仓位被淘汰。

**基本原则**：
- 单只股票仓位上限：20%（即使极度看好也不超过）
- 总仓位随市场状态动态调整：
  - 上升趋势：60-80%
  - 震荡市：30-50%
  - 下降趋势：0-20%（或空仓）
- 分批建仓原则：首次建仓不超过计划仓位的 1/3
- 止损设置：技术关键位下方（不超过总资金的 2% 风险）

### 8. 复盘框架（Post-Market Review）

**一句话**：不经过复盘的交易是重复犯同样的错误。

**每日复盘清单**：
1. 大盘状态：趋势方向？成交量是否正常？领涨板块？
2. 持仓评估：逻辑还在吗？止损位有没有被触发？
3. 操作回顾：今天执行纪律了吗？有没有情绪化操作？
4. 明日预案：关键位在哪里？什么条件下买/卖/不动？

**每周复盘清单**：
1. 本周净值变化 + 归因分析（选股贡献 vs 择时贡献 vs 运气）
2. 本周所有操作的统计：胜率、盈亏比、最大回撤
3. 有没有违反纪律的操作？原因是什么？如何避免？
4. 下周的观察清单 + 计划

## 决策检查清单（Decision Checklist）

下单前逐条核对：

1. 当前市场整体状态是什么？（趋势/震荡/单边下跌？）
2. 你的分析周期和操作周期是否一致？
3. 三面共振了吗？（技术面 + 基本面 + 资金面）
4. 止损位设在哪里？这个止损对应的亏损比例你能接受吗？
5. 仓位是否在规则范围内？
6. 你的买入逻辑是什么？这个逻辑在什么条件下不成立？
7. 有没有情绪驱动的成分？（FOMO、恐高、想回本？）
8. 同等条件下，有没有更好的选择？为什么选了这只？
9. 如果现在空仓，你还会买入吗？（帮助克服锚定效应）
10. 这一笔交易独立于上一笔，还是想"扳回"亏损？（警惕赌徒心态）
11. 对这个标的你做了足够的功课吗？还是听了别人推荐？
12. 今天是不是做了超出自己能力圈的事？
